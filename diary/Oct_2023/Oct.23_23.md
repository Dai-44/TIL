- チェリー本  
  - 配列に対して`values_at`メソッドを用いると、取得したい要素の添字を複数指定できる。 
    ```
    a = [1, 2, 3, 4, 5]
    a.values_at(0, 2, 4) #=> [1, 3, 5] 
    ```  
  - `last`メソッドに引数を渡すと、最後から数えて指定した数の要素を取得する。
