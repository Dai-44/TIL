### ゆる学び
- チェリー本  
  - `upto`メソッドは、レシーバの数字から引数で渡す数字まで数値を一つずつ増やしながら、同じ処理を繰り返したい場合に利用できる。`downto`メソッドは逆に、数値を一つずつ減らしながら処理を繰り返す。  
      ```
      a = []
      10.upto(15) { |n| a << n }
      
      a #=> [10, 11, 12, 13, 14, 15] 
      ```
